# Mentoring App

Платформа менторинга (MVP): каталог менторов, заявки, избранное, профили. Стек: React (Vite), NestJS, PostgreSQL.

- **Описание проекта и требований:** [progect.md](progect.md)
- **Запуск, CI, ngrok, типовые проблемы:** [docs/run.md](docs/run.md)
- **Модель веток (main / develop):** [docs/BRANCHES.md](docs/BRANCHES.md)

Быстрый старт: скопировать `backend/.env.example` → `backend/.env`, `frontend/.env.example` → `frontend/.env`, затем `docker-compose up -d` и открыть http://localhost:5173.
